# Nonlinear-Timeseries-Forecasting-Project
Nonlinear time series analysis in Python using mutual information, phase-space reconstruction, correlation dimension, Lyapunov exponent, and nearest-neighbor forecasting.

- Time-delay estimation using Average Mutual Information
- Phase-space reconstruction
- Correlation integral and correlation dimension analysis
- Maximal Lyapunov exponent estimation
- Nonlinear nearest-neighbor forecasting

Main results:
- Selected time delay: τ = 12
- Estimated maximal Lyapunov exponent: positive
- Last-value prediction error: 23.65%

Note:
The correlation dimension did not show clear saturation across the tested
embedding dimensions, so the embedding dimension result should be interpreted
with caution.
